## Codegen 当前产物

`generateFiles()` 当前输出：

- `types.ts`：GraphQL Code Generator 的 TypeScript 类型。
- `invalidation.ts`：导出 `Invalidation = GraphDataInvalidation`，和 runtime target 模型保持同名语义。
- `accessor.ts`：typed accessor、hooks/create functions、selector builders、schema contract、mutation descriptors。

其中真正驱动 runtime 行为的是 `accessor.ts` 内的 `gqlensSchema` 和 generated hook/create function。

`accessor.ts` 由 `code-block-writer` 生成。生成文件固定分为 typed nodes、schema contract、runtime entrypoints、selector builders、mutation descriptors，contract 容器按类型/字段展开，方便人和 LLM 快速定位 runtime 所需信息。

## 暂不实现 / 不应进入 Accessor API

- dynamic field API。
- list array API，例如 `.map()`、`[0]`、`.item()`。
- field-level directive / fetch policy chain。
- user-facing alias API。
- runtime `q.invalidate()`。
- accessor node enumeration / spreading as data object。
