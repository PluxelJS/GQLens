# 02. 字段访问

## Accessor Graph

Codegen 根据 GraphQL schema 生成 typed accessor。它**不是数据对象**，而是类型占位符——即使数据尚未返回，也能通过字段调用来声明 selection。

### Root Accessor

```ts
type QueryNode = {
  user(args: { id: string }): UserNode
  viewer(): UserNode
  todo(args: { id: string }): TodoNode
  todos(args?: TodoFilter): TodoListNode
}
```

### Entity Accessor

```ts
type UserNode = {
  id(): string
  name(): string
  avatar(): string | null
  online(): boolean
  posts(args?: PostArgs): PostListNode
}
```

### 使用示例

```ts
const q = useQuery()

const name   = q.user({ id: "1" }).name()   // → string
const avatar = q.user({ id: "1" }).avatar() // → string | null
```

一次调用同时产生：

```
记录 selection：Query.user(id:"1").name
读取 signal：  User:1.name
```

## 字段函数的两种形态

### 标量字段（终点）

```ts
function name(): string {
  ctx.select(path.field("name"))
  return ctx.cache.field(ref, "name").read()
}
```

### 关联字段（继续链式访问）

```ts
function posts(args?: PostArgs) {
  const nextPath = path.field("posts", args)
  ctx.select(nextPath)
  return createPostListNode(ctx, ref.relation("posts", args), nextPath)
}
```

## 列表访问模式

列表拆分为「身份」和「字段」两个层次：

```ts
// 获取 ID 列表
const ids = q.todos({ done: false }).ids()   // → string[]

// 按 ID 读取单个实体的字段
const title = q.todo({ id }).title()         // → string
```

推荐原则：

- 列表组件只订阅 ID 列表
- 行组件按 ID 读取字段
- 字段组件只订阅单个字段
- 不订阅整个对象
- 不返回临时对象作为响应式返回值

## Codegen 职责

Codegen 必须生成：

| 产物                   | 用途                               |
| ---------------------- | ---------------------------------- |
| Schema 类型             | GraphQL 类型在宿主语言的映射        |
| Root accessor          | 字段链式访问的入口                  |
| Entity node accessor   | 实体上每个字段的 typed 函数         |
| Args 类型              | 参数对象的类型定义                  |
| Entity identity 元数据 | 标识实体主键的字段（id/__typename）|
| Normalizer 元数据      | 响应数据到实体字段的映射规则        |
| Mutation wrapper 类型  | 带 optimistic 回调的 mutation 函数  |

其中 entity identity 元数据使 Planner 能自动补齐 `id` 和 `__typename`，用户无需手动声明。

## AST 编译插件（可选）

编译器插件纯粹是性能 / DX 优化层，**不是正确性的前提**。它可以做：

- 预计算 selection path
- 生成稳定的 operation name
- 提取变量形状
- 生成 persisted query hash
- 开发期静态诊断非法字段访问
- tree-shaking 辅助
